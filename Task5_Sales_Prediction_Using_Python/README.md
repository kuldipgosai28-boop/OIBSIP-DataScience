# Sales Prediction Using Python

## Objective

The objective of this project is to predict product sales based on advertising expenditure and other influencing factors using machine learning techniques.

## Dataset

The dataset contains information about advertising budgets spent on TV, Radio, and Newspaper along with corresponding sales figures.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Sales Prediction

## Machine Learning Model

A regression model was developed to predict future sales based on advertising data.

## Key Analysis Performed

* Data Cleaning
* Correlation Analysis
* Data Visualization
* Feature Selection
* Model Training and Testing
* Performance Evaluation

## Results

The model successfully identified the relationship between advertising expenditure and product sales, enabling accurate sales predictions.

## Conclusion

This project demonstrates how machine learning can help businesses forecast sales and make data-driven marketing decisions.
