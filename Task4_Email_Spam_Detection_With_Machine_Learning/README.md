# Email Spam Detection with Machine Learning

## Objective

The objective of this project is to build a machine learning model that can classify emails as Spam or Ham (Not Spam) based on their content.

## Dataset

The dataset contains email messages labeled as spam or ham for training and testing the machine learning model.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* NLP (Natural Language Processing)

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction
5. Model Training
6. Model Evaluation
7. Spam Prediction

## Machine Learning Model

A classification model was trained to identify whether an email is spam or not.

## Key Analysis Performed

* Text Cleaning
* Tokenization
* Stopword Removal
* Feature Extraction using TF-IDF
* Model Training and Testing
* Accuracy Evaluation

## Results

The model successfully classified email messages into spam and non-spam categories with high accuracy.

## Conclusion

This project demonstrates the use of machine learning and natural language processing techniques for detecting spam emails and improving email security.
